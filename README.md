# 🌐 SocialBook – Mini Social Network Web App

SocialBook is a simple and modern mini social networking web application where users can create posts and interact through a clean and user-friendly interface.

## 🚀 Features

* 🧑 User name input system
* ✍️ Create and share posts
* 📰 Dynamic social feed display
* 🎨 Modern and responsive UI design
* 📂 Organized project structure (MVC style)
* ⚡ Fast and lightweight frontend

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Node.js
* Express.js

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/KASHIF-80/socialbook-web.git
```

2. Open project folder

```
cd socialbook-web
```

3. Install dependencies

```
npm install
```

4. Start server

```
node server.js
```

5. Open browser

```
http://localhost:5500
```
